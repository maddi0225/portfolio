Practice Makes

Perfect!

A good website starts with a good design. I put all of my effort into making a functional website with designs that catch attention and please the eye.

CSS

I use CSS to make personalized schedules. This includes grids, color coding, images, and links to helpful websites. It is a great way to keep track of your week, especially for those in school. I also use CSS to help design websites, which can be seen in other projects.

Bootstrap

To assist with webdesign, I have become familiar with the bootstrap framework. This allows for easier customization and a prettier website. I used this framework to make my website Breadbox. Breadbox allows users to sign up to recieve recipes and tips for cooking. It's like redbox but for bread!

Figma

Figma is used to created the prototype for app design. This includes the different page designs, color schme, and functionality between pages. I used Figma to create the protype of the app Ouchie Go Bye-Bye. It is an app geared towards parents to give information about a variety of different symptoms of illness and injury. It also allows for discussion between app members to ask questions and get helpful answers!
